#### STL VIEWER is an web based STL File viewer which is free to use and need no sign in , etc . We can Chnage colour of the model see it in mesh view ..

---

## Inspo:
My inspo was some of the popular website to see STL viewer which I use daily and they are good

## Why I built it:
Most of the web STL Viewerare good but there are too much Ads , sign in , log in - required just to view an STL File So I make one where there will be no Ads , sign in , log in - required to
view the File and there are more features than an usual website and I have built it and I can show it someone ...

  |:--|
  | HTML |

## Features

* MODEL STATS ( WEIGHT , VOLUME , SURFACE AREA , TRIANGLES , PRINT TIME )
* DIMENSIONS
* FILE INFO
* MATERIAL COLOR ( Can be changed )
* BACKGROUND ( Colour Can Be Changed )
* MODEL ROTATION
* CAMERA PRESET ( Like top view , bottom view , etc )

---

## To run it locally 
* 1) Fork the repo 
* 2) Open the file in your code editor (VS Code )
* 3) Open The Terminal 
* 4) Write - open index.html (Mac) , start index.html (Windows) , xdg-open index.html (Linux)
